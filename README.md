# Maven repository cli

[![Build Status](https://travis-ci.org/clemstoquart/maven-repo-cli.svg?branch=master)](https://travis-ci.org/clemstoquart/maven-repo-cli)

## Requirements

Nodejs 10 is required.

## Setup

    npm i
    
## Make it a regular cli tool

    npm link
    
Then

    maven-repo-cli --help
